Every PM I know is scrambling to learn AI tools they think will set them apart. This is a trap. 

It’s 2026. Knowing how to use agent harnesses and integrate models into products is now table stakes. You need to know how to use Claude Code, Cursor, NotebookLM, and whatever new tool ships next month, just like knowing how to use Google Docs, spreadsheets, and Figma.

But nobody has ever gotten hired as a PM because they were great at Jira.

I love Shreyas Doshi‘s take on this: when everyone has access to the same tools and models, the differentiator is Product Sense. PMs need to hone in on empathy, strategic thinking, taste, the ability to predict what will actually work for users and move the needle. 

Today’s PM interviews are indeed assessing that you know how to use AI tools and frameworks. But tools alone will not get you hired. You need to show you can ship something people want. 

Learn the tools. But don’t confuse the instrument for the skill.
